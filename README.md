# Hotel Booking API

## Overview

Hotel Booking API is a RESTful ASP.NET Core Web API developed in C#.

The application allows management of hotels, rooms, customers and reservations.

## Features

* Hotel Management
* Room Management
* Customer Management
* Reservation Management
* Authentication (JWT Ready)
* Swagger Documentation
* Entity Framework Core
* SQLite Database

## Technologies

* ASP.NET Core Web API
* C#
* Entity Framework Core
* SQLite
* Swagger/OpenAPI

## Project Structure

* Controllers
* Models
* DTOs
* Data Layer
* Services
* Authentication

## Main Endpoints

### Hotels

GET /api/hotels

POST /api/hotels

### Rooms

GET /api/rooms

POST /api/rooms

### Customers

GET /api/customers

POST /api/customers

### Reservations

GET /api/reservations

POST /api/reservations
